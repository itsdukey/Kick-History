# Kick History
![RuneLite Plugin Hub](https://img.shields.io/badge/RuneLite-Plugin_Hub-orange)
![License](https://img.shields.io/badge/License-BSD--2--Clause-blue)
![Java](https://img.shields.io/badge/Java-17%2B-red)

A simple RuneLite plugin that logs when players are kicked from a Friends Chat. It shows a list in the sidebar and can optionally send the logs to a Discord channel.

## Features
* **Sidebar Log:** Displays the time and username of everyone kicked during your session.
* **Discord Webhooks:** Sends kick notifications to Discord (includes the name of the moderator who did the kick).

## How to use
1. Open the **Kick History** settings in RuneLite.
2. If you want Discord logs, paste your **Discord Webhook URL** into the config and toggle it on.
3. Use the **Copy All** button in the sidebar if you need to save the log to a text file.

## Why this exists
in busy chats sometimes you need to kick multiple people before you are able to sit and log the kicks properly.
This plugin helps log the exact usernames and time of the kick, so admins can properly log kick history and repeated offenders.
